# 주가 데이터 클러스터링을 통한 종목 군집화 및 해석

## 개요/목표

- 종목을 ‘유사한 움직임/특성’ 기준으로 군집화하고, 군집별 특징을 해석합니다.
- 차원축소 + 군집 + 해석(대표 종목, 특징량)을 하나의 스토리로 구성합니다.

## 데이터 범위(예시)

- 입력 특징 예시: 최근 N일 수익률 시계열, 변동성, 거래량 패턴, 기술지표 요약 통계
- 군집 알고리즘: K-means, GMM, Agglomerative, DBSCAN 등

## 권장 진행 단계

1. 군집화에 사용할 특징 정의(시계열 그대로 vs 요약 통계)
2. 스케일링/정규화 및 이상치 처리
3. 차원축소(PCA/UMAP/t-SNE)로 구조 파악
4. 군집화 수행 + 군집 수/파라미터 선택 근거 제시
5. 군집 해석: 군집별 평균 패턴, 섹터 분포, 대표 종목

## 필수 산출물

- 군집 결과(라벨) 포함 데이터셋
- 시각화(2D 임베딩, 군집별 패턴 플롯) 및 해석 리포트
- 군집 안정성/재현성 점검 결과

## 평가/검증 포인트

- 군집 수 선택의 근거(실루엣, BIC/AIC, 안정성 등)
- 기간 변경 시 군집이 유지되는지(robustness)
- 군집의 경제적 의미(섹터/이벤트/공통 요인) 설명

## 확장 아이디어(선택)

- 시장 레짐별 군집 변화 분석
- 동적 클러스터링(rolling window)으로 군집 전이 추적
- 군집을 피처로 사용해 예측 모델 성능 개선
